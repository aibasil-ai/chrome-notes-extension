輕筆記隱私權政策

本政策說明「輕筆記」Chrome 擴充功能在你使用服務時，如何蒐集、使用、儲存與保護資料。 使用本擴充功能即表示你理解並同意本政策內容。

隱私權政策條款

我們以最小必要原則處理資料。你建立的筆記內容主要保存在瀏覽器儲存空間中，若你啟用 Chrome Sync，資料將依 Google 帳號同步機制在你的裝置間同步。

1. 蒐集哪些資料

為提供筆記功能，系統可能處理以下資料：

  • 你輸入的筆記內容、標題、標籤與建立時間。
  • 你手動儲存的來源網址與頁面標題。
  • 必要的設定資料，例如編輯偏好與排序方式。

2. 資料使用目的

資料僅用於提供與改善核心功能，不作其他用途：

  • 顯示、編輯、搜尋與分類你的筆記。
  • 在你啟用同步時維持跨裝置資料一致性。
  • 支援 JSON 匯出與匯入，方便你自行備份與遷移。

3. 儲存與同步機制

擴充功能使用 Chrome Extension 儲存 API：

  • 本機資料可儲存在 `chrome.storage.local`。
  • 啟用同步時可使用 `chrome.storage.sync`。
  • 我們不另行建立獨立後端伺服器保存你的筆記。

4. 第三方服務

本擴充功能可能依賴以下第三方平台能力：

  • Google Chrome 提供的擴充功能執行與儲存環境。
  • Chrome Sync（由 Google 帳號與 Google 服務管理）。
  • 我們不出售、出租或分享你的筆記資料給廣告商。

5. 你的權利與控制

你可透過以下方式管理個人資料：

  • 在擴充功能中刪除單筆或全部筆記資料。
  • 使用 JSON 匯出保留備份，或匯入先前資料。
  • 移除擴充功能以停止後續資料處理。

6. 政策更新

若本政策有重大變更，將於本頁更新版本與日期。

  • 更新後政策自公告日期起生效。
  • 建議你定期檢視本頁，掌握最新內容。

若你不同意本政策內容，請停止使用本擴充功能，並刪除本機或同步中的相關資料。

隱私權相關問題與聯絡方式

若你對資料處理方式有疑問，請透過 Chrome 線上應用程式商店中的開發者聯絡資訊與我們聯繫。
