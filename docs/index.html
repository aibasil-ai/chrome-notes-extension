核心功能

以下模組對應你日常的記錄、整理、檢索與備份流程，讓資訊層次更清楚、回查更有效率。

三介面切換

依任務深度切換操作視角，不需要為了整理內容而離開原始頁面。

雙編輯模式

支援 Markdown 與純文字，讓短記錄與長文整理在同一工具完成。

標籤 + 全文搜尋

可同時搜尋標題與內容，再搭配標籤縮小範圍，提高回查速度。

同步與容量控制

近期資料同步到 Chrome Sync，並提供空間監測避免超限。

JSON 匯出匯入

定期備份與資料搬遷更穩定，匯入時可合併既有筆記內容。

來源脈絡保存

可附加 URL、頁面標題與時間戳，回顧時保留完整研究上下文。
